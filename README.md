# Vaadin_学習教材

本プロジェクトは、千歳科学技術大学のプロジェクトメンバー用の講習資料である。

研究で独自開発するシステムのベースに使うJava Webフレームワークの組み合わせの中で、ユーザ作成機能、ログイン機能の実装を体験的に練習する。

これにより、クラシックなサーバサイドWebアプリケーションの作り方を学ぶ。
